-El duende chapucero


El Duende Chapucero (Goblin Tinkerer, en ingles) es un NPC que vende objetos para construcción. Para encontrarlo primero debes vencer al Ejército de duendes y buscarlo en las cuevas de esqueletos para desatarlo y tener una casa disponible. Este se defiende utilizando Bola con pinchos, Puede ser mas fácil de buscar con el analizador de vidas.

Tiene la opción de Reforjar, lo que permite darle o cambiar los bonos a tus Armas y Accesorios.
